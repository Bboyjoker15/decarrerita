Detalles:
  - Modificacion de docker-compose.yml: Se comento la directiva 
    'network_mode: host' y se agrego explicitamente el mapeo de 
    puertos '5432:5432' para resolver el error P1001 (Can't reach 
    database) al intentar correr las migraciones desde la consola 
    nativa de Windows.
  - Sincronizacion de Base de Datos: Se ejecutaron exitosamente las 
    migraciones de Prisma para crear las 12 tablas en el contenedor 
    y se corrio el comando de seed (npx prisma db seed) para poblar 
    el sistema con los usuarios y bancos de prueba.
  - Inicializacion del Frontend: Creacion manual de la carpeta 'src/' 
    junto con sus archivos base (main.jsx e index.css configurado con 
    las directivas basicas de TailwindCSS).
  - Desarrollo del modulo de Autenticacion: Implementacion de un 
    formulario de Login reactivo en App.jsx con persistencia temporal 
    del JWT en localStorage y renderizado condicional para mostrar el 
    estado "Sesion iniciada" y el rol exacto de la cuenta conectada.
  - Correccion de Payload: Se ajusto el envio de datos en Axios para 
    cambiar la clave 'email' por 'correo', adaptando la peticion a los 
    requerimientos estrictos de express-validator en el backend.
  - Resolucion de Bug Critico (CORS / Preflight): Se soluciono un 
    bloqueo de carga infinita en el navegador provocado por la peticion 
    OPTIONS (Preflight) que quedaba en estado pending. Se agrego un 
    middleware nativo en backend/src/app.js para inyectar las cabeceras 
    de control de acceso (Access-Control-Allow-Origin) y responder 
    inmediatamente con estatus 200 a los metodos OPTIONS.
  - Verificacion de integracion: Flujo completo probado de forma local 
    con exito usando las credenciales de los 4 roles del sistema.
